## How to Use After GitHub Publication

### For End Users:

1. **Clone repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/2xkoGPAnalyzer_VisualStudio.git
   cd 2xkoGPAnalyzer_VisualStudio
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Place video in Desktop:**
   ```
   Desktop/YourVideo.mp4
   ```

4. **Run analysis (choose one):**
   ```bash
   # Main project (recommended):
   python analyze_first_match.py
   
   # OR CODEX_CHATGPT (advanced):
   python CODEX_CHATGPT/run_codex_analyzer.py
   ```

5. **View results:**
   - Main: Check `output/Blitz_vs_Blitz_Juggernaut_Analysis.html`
   - CODEX: Check `CODEX_CHATGPT/output/blitz_mirror_report.html`
